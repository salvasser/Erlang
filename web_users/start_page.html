Welcome!


This is the start page of the web application.


This application has the following features:


Displaying users from the database (/get-users)


Displaying a specific user by id (/get-user/id-X), where X is the id number


Removing a user from the database, you need to specify an id (/delete-user/id-X), where X is the id number


Adding a user to the database (/add-user)
